The Light

This game was built as a part of Berlin Mini Game Jam, July 2013 in roughly 7 hours.

The goal is to step into a door without touching darkness.

Controls:

  • Movement: arrow keys
  • Rotate light: left/right
  • Abandon light: up
  • Continue after death / entering a door: any arrow key
